# 🗳 AI Voting System

## 📌 Description
A secure role-based online voting system built using Node.js, Express, and MongoDB.

---

## 🚀 Features
- Admin Authentication
- Voter Authentication
- JWT-based login system
- Prevent double voting
- MongoDB database integration
- AES encryption (planned)

---

## 🛠 Tech Stack
- Node.js
- Express.js
- MongoDB
- JWT

---

## ▶️ How to Run

```bash
npm install
npm start
```
